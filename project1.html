Gone Country

Github Live
SUMMARY

Gone country is a country music blog website. The client goes by the pen name The Armadillo. The author has the ability to create/edit/delete posts as well as upload images to their posts.

The client was very specific about privacy so I am unable to show the original project repo. I was given permission to duplicate the project and show a near identical project I named Country Clue. For code examples please see the github link listed above.


TECH

  • MERN
  • MongoDB, Express, ReactJs, NodeJs


FEATURES

This project uses restful routing, nosql data models with MongoDb, image upload capability via the cloudinary api. The author has the ability to add tags to posts to indicate the content of the article. A user can also 'like' an article posted by the author.
