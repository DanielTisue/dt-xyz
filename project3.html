SUMMARY

OlivieriPhoto is an artist photography portfolio website. This project was created to fufill a need for client who is pursuing artisitc photography. Originally, the client was using a popular website builder but did not find value in the monthly reoccuring costs it imposed. The client expressed wanting to, not only more specifically taylor the design of the site, but also to cut monthly cost significantly.

This site was originally created in 2018-2019 but went through a massive re-design in 2021 @ the clients request.


TECH

  • Html, Css, EJs, Javascript
  • Node.Js, MongoDb, Express


FEATURES

This project uses restful routing, nosql data models with MongoDb, nodemailer for contact method, user authentication and middleware security, async, and image upload capability.
